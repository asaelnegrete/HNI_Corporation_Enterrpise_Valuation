# Enterprise Valuation of HNI Corporation

## Overview

This project presents a comprehensive enterprise valuation of HNI Corporation using discounted cash flow (DCF) analysis, comparable company analysis, and financial forecasting. The objective was to estimate HNI's intrinsic value and evaluate strategic capital allocation decisions based on projected operating performance.

---

## Project Objective

Develop a three-year financial forecast and estimate HNI Corporation's enterprise value using industry-standard valuation techniques.
